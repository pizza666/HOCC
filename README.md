# Halls of Corruption

An RPG/dungeon crawler for C64 with pseudo 3D graphics 
Rewritten in C
Original ASM project from https://github.com/pizza666/HOC/

## Build instructions

Needs cc65 V2.19 - https://cc65.github.io/

1. Compile with:
make all

2. Create diskimage with all data with:
make diskimage
